# Online Learning Behaviour Analysis Dashboard

## 1. Project Overview

This project analyzes **online learning behaviour** using a dataset containing student demographics, engagement metrics, and course performance indicators. The dashboard is created using **Microsoft Power BI** to visualize patterns in student participation, study habits, and course completion.

The goal of the project is to understand how students interact with online learning platforms and identify factors affecting engagement, performance, and satisfaction.

---

## 2. Objectives

The main objectives of this project are:

* Analyze **student engagement** based on learning hours, quizzes attempted, and assignments submitted.
* Track **course completion rates and student satisfaction**.
* Compare **learning platforms, devices, and learning modes**.
* Identify **student drop-off points** during the learning process using a funnel chart.
* Visualize trends and patterns using interactive dashboards.

---

## 3. Dataset Description

The dataset contains information about students participating in online learning platforms.

### Dataset Columns

* Student_ID – Unique identifier for each student
* Country – Student’s country of residence
* Age – Age of the student
* Gender – Gender of the student
* Education_Level – Highest education level completed
* Field_of_Study – Academic discipline of the student
* Platform_Used – Online learning platform used
* Device_Used – Device used to access the course
* Learning_Mode – Type of learning (Self-paced, Instructor-led, Hybrid)
* Enrollment_Date – Date the student enrolled in the course
* Daily_Learning_Hours – Average number of hours spent learning per day
* Quizzes_Attempted – Number of quizzes attempted by the student
* Assignments_Submitted – Number of assignments submitted
* Course_Completion_Rate – Percentage of course completed
* Satisfaction_Rating – Student satisfaction score

---

## 4. Key Performance Indicators (KPIs)

The dashboard includes the following KPIs:

* Total Students
* Average Daily Learning Hours
* Average Course Completion Rate
* Average Satisfaction Rating
* Total Quizzes Attempted
* Total Assignments Submitted

---

## 5. Dashboard Visualizations

The dashboard includes several visualizations to analyze the data:

### Charts and Visuals

* KPI Cards – Display key metrics such as total students and average completion rate
* Donut Chart – Distribution of students by education level
* Column Charts – Students by platform, learning mode, and field of study
* Line Chart – Enrollment trends and learning activity over time
* Scatter Plot – Relationship between learning hours and completion rate
* Funnel Chart – Student progression from enrollment to course completion

---

## 6. Filters and Slicers

Interactive slicers are included to allow dynamic filtering of the dashboard:

* Country
* Gender
* Education Level
* Platform Used
* Learning Mode
* Device Used
* Enrollment Date

---

## 7. Tools Used

* Microsoft Power BI – Data visualization and dashboard development
* Dataset (CSV/Excel format) – Online learning behaviour data

---

## 8. Insights Expected

This dashboard helps to identify:

* Which platforms and devices are most commonly used by students
* The relationship between learning hours and course completion
* Student engagement patterns across different fields of study
* Stages where students drop out during the course

---

## 9. Conclusion

The Online Learning Behaviour Dashboard provides valuable insights into student engagement and learning patterns. These insights can help educators and platform administrators improve course design, enhance student participation, and increase course completion rates.
